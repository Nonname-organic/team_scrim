# AXELIA Analytics データ利用ポリシー

本データ利用ポリシー（以下「本ポリシー」といいます。）は、AXELIA（以下「当チーム」といいます。）が提供する「AXELIA Analytics」（以下「本サービス」といいます。）において取り扱う登録データの取扱方針を定めるものです。本ポリシーは、AXELIA Analytics 利用規約の一部を構成します。
第1条（目的）
本ポリシーは、本サービスにおいて当チームが取り扱う登録データについて、その所有権、利用範囲、管理方法及び削除等の取扱いを明確にし、利用者が安心して本サービスを利用できるようにすることを目的とします。

# 第2条（取得するデータ）

当チームは、本サービスの提供のために、次のデータ（総称して「登録データ」といいます。）を取り扱います。
1. 試合分析、ラウンド分析等のために利用者が入力・登録する試合関連データ
2. プレイヤー分析のために利用者が入力・登録するプレイヤー関連データ
3. 統計表示、ダッシュボード閲覧及びレポート出力のために本サービスが登録データを処理して生成する分析結果
4. 本サービスの利用に伴い生成される操作履歴・設定情報等の付随データ

# 第3条（データの取得方法）

当チームは、登録データを、原則として利用者による本サービスへの入力・アップロード・操作を通じて取得します。当チームは、利用者の同意なく、利用者のチームに関するデータを外部から独自に収集することはありません。

# 第4条（利用目的）

1. 当チームは、登録データを、次の目的の範囲内でのみ利用します。
    1. 本サービスの提供（試合分析、プレイヤー分析、ラウンド分析、統計表示、ダッシュボード閲覧、レポート出力等の機能提供）
    2. 本サービスの保守、障害対応及び品質維持
    3. 本サービスの改善及び新機能の開発
    4. 利用者からの問い合わせ対応
2. 当チームは、本サービスの改善のために登録データを利用する場合であっても、特定のチーム又は個人を識別できない形で統計的に処理する等、利用者の正当な利益に配慮した方法によるよう努めます。
3. 当チームは、他チームの登録データを、本サービスの提供・保守・改善という目的の範囲を超えて分析・利用しません。
4. 当チームは、利用者データを、AXELIA Analyticsのサービス提供、分析機能の提供、サービス品質の維持および改善、不具合の調査・対応、セキュリティ対策の実施その他本サービスの適切な運営に必要な範囲で利用します。
5. 利用者の個別データを、利用者の同意なく第三者へ提供することはありません。ただし、法令に基づく場合その他法令により認められる場合を除きます。

# 第5条（データの所有権）

1. 登録データに関する権利は、当該データを登録した利用者（チーム）に帰属します。当チームは、登録データの所有権を取得するものではありません。
2. 当チームは、本サービスの提供・保守・改善に必要な範囲に限り、利用者から登録データの利用の許諾を受けるものとします。この許諾は、本サービスの提供に必要な限度でのみ効力を有します。
3. 利用者が本サービスへ登録、入力またはアップロードしたデータの権利は、当該利用者または当該利用者が所属するチームその他正当な権利者に帰属します。
4. 当チームは、本サービスの提供、保守、改善および運営に必要な範囲でのみ当該データを利用します。

# 第6条（データの分離管理・アクセス制御）

1. 当チームは、チームごとに登録データを分離して管理します。ある利用者は、自己のチームに割り当てられたデータのみを閲覧・編集でき、他チームの登録データを閲覧・編集することはできません。
2. 当チームは、登録データへのアクセスを、業務上必要な当チームの従業者等に限定し、アクセス権限を適切に管理します。
3. 本条に関する技術的・組織的な安全管理措置の詳細は、セキュリティポリシーに定めるところによります。

# 第7条（保存期間）

1. 当チームは、登録データを、利用者が本サービスを利用している期間中保存します。
2. 利用契約の終了後は、当チームは、法令上必要な場合又は当チームが定める合理的な期間（バックアップ上の保存を含みます。）を経過した後、登録データを削除又は復元できない形で消去します。

# 第8条（第三者提供）

1. 当チームは、次のいずれかに該当する場合を除き、登録データを第三者に提供しません。
    1. 利用者の同意がある場合
    2. 法令に基づく場合
    3. 人の生命、身体又は財産の保護のために必要があり、利用者の同意を得ることが困難な場合
    4. 本サービスの提供に必要な範囲で、後記第9条〔※委託に関する記載はプライバシーポリシー第7条を参照〕に従い業務を委託する場合
2. 当チームは、利用者の個別の登録データ又は分析結果を、利用者の同意なく公開しません。
3. 当チームは、本サービスの提供に必要な範囲で、クラウドサービス事業者等に登録データの保存・処理を委託することがあります。この場合、当チームは、委託先に対し必要かつ適切な監督を行います。

# 第9条（データの削除）
1. 利用者は、当チーム所定の方法により、自己のチームに関する登録データの削除を請求できます。
2. 当チームは、前項の請求を受けた場合、法令上保存が義務付けられている場合等を除き、合理的な期間内に対象の登録データを削除します。
3. バックアップ等に一時的に残存するデータについても、当チームは、通常の運用サイクルの中で削除又は消去されるよう管理します。

# 第10条（利用者の権利）

利用者は、当チーム所定の方法により、自己のチームに関する登録データについて、閲覧、エクスポート（当チームが提供する形式による出力）及び削除を請求することができます。当チームは、これらの請求に対し、合理的な範囲で対応します。

他チームのデータは、本サービス上で提供される分析・比較機能の利用に限り利用できます。
利用者は、他チームのデータまたは分析結果を、本サービス外へ転載、複製、再配布、販売、第三者への提供、生成AIその他の機械学習モデルの学習への利用、スクレイピングその他本サービスの利用目的を超えて利用してはなりません。

# 第11条（個人情報の取扱い）

1. 登録データに個人情報（プレイヤー個人を識別できる情報を含む場合があります。）が含まれる場合、当該個人情報の取扱いについては、プライバシーポリシーの定めが適用されます。
2. 利用者が本サービスにプレイヤー等の個人情報を登録する場合、利用者は、当該個人情報の取得及び当チームへの提供について、適用法令上必要な範囲で適法な根拠を有していることを確認するものとします。
