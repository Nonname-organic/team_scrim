# AXELIA Analytics プライバシーポリシー

AXELIA（以下「当チーム」といいます。）は、「AXELIA Analytics」（以下「本サービス」といいます。）の提供にあたり取得する個人情報を、個人情報の保護に関する法律（以下「個人情報保護法」といいます。）その他の関係法令を遵守して取り扱います。本プライバシーポリシー（以下「本ポリシー」といいます。）は、AXELIA Analytics 利用規約の一部を構成します。

# 第1条（基本方針）

当チームは、個人情報の重要性を認識し、その適正な取扱いを確保するため、関係法令及び本ポリシーを遵守し、個人情報の保護に関する合理的な安全管理措置を講じるとともに、その継続的な改善に努めます。

# 第2条（取得する個人情報）

当チームは、本サービスの提供に関し、次の個人情報を取得します。

1. アカウント登録時に利用者が入力する情報（チームの担当者・管理者の氏名、メールアドレス、その他連絡先等）
2. お問い合わせ時に本人が入力する情報（氏名、メールアドレス、問い合わせ内容等）
4. 本サービスの利用に伴い自動的に取得される情報（第4条のCookie等の情報及び第5条のアクセスログ）
4. 登録データに含まれる個人情報（利用者がプレイヤー分析等のために登録する情報で、特定の個人を識別できるものが含まれる場合があります。）

当チームは、本条に列挙した以外の方法で個人情報を取得することはありません。

# 第3条（利用目的）

当チームは、取得した個人情報を、次の目的の範囲内で利用します。

1. 本サービスの提供、維持及び本人・利用者の認証のため
2. 利用者への連絡、通知及び問い合わせ対応のため
3. 本サービスの保守、不正利用の防止及びセキュリティの確保のため
4. 本サービスの品質向上及び改善のため
5. 利用規約違反への対応のため
6. 法令に基づく対応のため

# 第4条（Cookie等の利用）

当チームは、本サービスの利便性向上、セッションの維持、不正利用の防止及び利用状況の把握のために、Cookie及びこれに類する技術を利用することがあります。
利用者は、ブラウザの設定によりCookieの受け入れを拒否することができますが、その場合、本サービスの一部機能が利用できないことがあります。

※ 外部の解析ツール等を利用する場合は、当該ツール名・目的・オプトアウト方法を本条に追記してください。

# 第5条（アクセスログの取得）

当チームは、本サービスの運用・保守、セキュリティの確保及び品質改善のため、本サービスへのアクセスに関する情報（IPアドレス、ブラウザの種類、アクセス日時、操作履歴等）を自動的に取得し、記録することがあります。

# 第6条（保存期間）

当チームは、個人情報を、利用目的の達成に必要な期間又は法令で定められた期間、保存します。当該期間を経過した個人情報は、遅滞なく削除又は消去するよう努めます。登録データに含まれる個人情報の保存期間は、データ利用ポリシーの定めに準じます。

# 第7条（安全管理措置）

1. 当チームは、取り扱う個人情報の漏えい、滅失又は毀損の防止その他の個人情報の安全管理のために、組織的、人的、物理的及び技術的な合理的安全管理措置を講じます。
2. 前項の安全管理措置の詳細は、セキュリティポリシーに定めるところによります。
3. 当チームは、安全管理措置について、継続的な見直し及び改善に努めます。

# 第8条（第三者提供・委託）

1. 当チームは、次のいずれかに該当する場合を除き、あらかじめ本人の同意を得ることなく、個人情報を第三者に提供しません。
    1. 法令に基づく場合
    2. 人の生命、身体又は財産の保護のために必要があり、本人の同意を得ることが困難な場合
    3. 公衆衛生の向上又は児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難な場合
    4. 国の機関等が法令の定める事務を遂行することに対して協力する必要がある場合
2. 当チームは、利用目的の達成に必要な範囲で、個人情報の取扱いの全部又は一部を、クラウドサービス事業者等の委託先に委託することがあります。この場合、当チームは、委託先に対し必要かつ適切な監督を行います。

# 第9条（開示・訂正・利用停止等の請求）

1. 本人は、当チームに対し、個人情報保護法の定めに従い、自己の個人情報の利用目的の通知、開示、内容の訂正・追加・削除、利用の停止・消去及び第三者提供の停止（以下「開示等」といいます。）を請求することができます。
2. 開示等の請求は、当チーム所定の方法により受け付けます。当チームは、本人からの請求であることを確認したうえで、法令に従い、合理的な期間内に対応します。
3. 登録データに含まれる個人情報について本人から開示等の請求があった場合、当チームは、必要に応じて当該データを登録した利用者と連携して対応することがあります。

# 第10条（プライバシーポリシーの変更）

当チームは、法令の改正又は本サービスの内容の変更等に伴い、本ポリシーを変更することがあります。変更後の本ポリシーは、当チームが本サービス上又は当チームウェブサイト上に掲示した時点から効力を生じます。重要な変更を行う場合は、当チームは、合理的な方法により周知します。

# 第11条（お問い合わせ・苦情の申出先）

個人情報の取扱いに関するお問い合わせ、開示等の請求及び苦情は、次の窓口までご連絡ください。
